# Contributing

We welcome contributions to ProxyAuthK8s! Whether you're fixing bugs, adding new features, or improving documentation, your help is appreciated.

The base specification of the [project can be found here](https://batleforc.github.io/weebo-si/0.projects/reverse-api-kube-oidc-based.html).

For further information on how to contribute, please refer to the [Contributing Guide](https://batleforc.github.io/ProxyAuthK8S/docs/developper-guide)
